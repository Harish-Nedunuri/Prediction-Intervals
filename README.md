## Development

To contribute to this library, first checkout the code. Then create a new virtual environment:

    cd prediction-intervals
    python -m venv venv
    source venv/bin/activate

Now install the dependencies and test dependencies:

    pip install -e '.[test]'

To run the tests:

    pytest
